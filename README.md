# RuneQuest Combat Calculator

A combat tracker and calculator for the RuneQuest RPG system.

## Features

- **Hit Location System**: D20-based hit location tracking with armor protection by location
- **Character Stats**: Track STR, DEX, CON, SIZ, weapon skills
- **Armor by Location**: Head, arms, legs, chest, abdomen - each with individual armor points
- **Weapon Library**: Broadsword, spear, battle axe, composite bow, javelin, dagger
- **Damage Calculation**: Automatic damage bonus calculation based on STR + SIZ
- **Combat Modifiers**: Difficulty adjustments and situational modifiers
- **Special Maneuvers**: RuneQuest-specific combat actions

## Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Single HTML file - no build process required

## Local Development

Simply open `index.html` in a web browser. No installation or build process needed.

## Deployment

This is a static HTML file and can be deployed to any static hosting service:
- Netlify
- Vercel
- Cloudflare Pages
- GitHub Pages

## RuneQuest System

RuneQuest uses a percentile skill system with location-based damage. This calculator helps track:
- Character attributes (STR, DEX, CON, SIZ)
- Hit locations (1-20 on d20)
- Armor points by location
- Weapon damage and bonuses
- Effective skill after modifiers
